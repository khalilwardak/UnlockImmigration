# Quick Start Guide

## Get Your Website Live

### 1. Customize Content
Edit `index.html` to update personal information, services, and contact details.

### 2. Deploy to GitHub Pages
1. Create a public GitHub repository
2. Upload files: index.html, styles.css, script.js
3. Go to Settings → Pages
4. Select "Deploy from a branch" → main branch
5. Save

### 3. Connect Custom Domain
1. Add CNAME record: www → your-username.github.io
2. In GitHub Settings → Pages, add your custom domain
3. Enable HTTPS

## Form Integration
Set up Formspree or EmailJS to make the contact form functional.
