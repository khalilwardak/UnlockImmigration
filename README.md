# Professional Immigration Consulting Website

A modern, responsive static website for immigration consulting services.

## Features

- Fully responsive design
- Modern UI with smooth animations
- SEO optimized
- Fast loading
- Easy to customize

## Quick Start

1. Clone or download this repository
2. Customize content in `index.html`
3. Deploy to GitHub Pages or any static hosting

## Customization

Edit `index.html` to update:
- Personal information
- Services offered
- Contact details
- Business hours

Edit `styles.css` to customize:
- Colors (CSS variables section)
- Fonts
- Layout spacing
